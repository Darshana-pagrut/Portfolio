Web Design

Date: 16th Jan 2023 - 22th Jan 2023


Objective:


To educate participants on designing, developing and deploying a website.


Description:


In this module participants learned from seniors, who provide us with a comprehensive understanding of web development. The trainers cover topics related to front-end development. The topics covered include HTML, CSS, javascript git, github, mongodb and two mini projects- 1.Calculator 2.Personal Portfolio Website.
This module was a great way to enhance Web Development skills. The event offers participants an immersive, hands-on learning experience, access to cutting-edge tools and technologies, and networking opportunities. By partnering with senior web developers, participants gained the knowledge we need to build professional-grade websites.


Expected Outcomes:


Building a functional website from the scratch.
