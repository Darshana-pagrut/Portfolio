Introduction:

I, a proud member of the FabLab community at SSGMCE, embarked on a transformative learning journey through various modules. The first module, focusing on web development, offered an intensive one-week exploration of the fundamentals of frontend web development, including HTML, CSS, Git, GitHub, and the basics of JavaScript. Week-Long Learning Process: The web development module began with an immersive week of teaching and learning. I was introduced to the core technologies that power the frontend of websites. The module's structured curriculum covered HTML, the foundation of web content; CSS, the language of style and layout; and the powerful version control system, Git, along with its collaborative platform, GitHub. Additionally, I delved into the basics of JavaScript, gaining insights into how interactivity and dynamic behavior are introduced to web pages.

Hands-on Project:

Building a Simple Calculator:

One of the highlights of this module was the hands-on project that allowed me to apply the concepts I had learned. I was tasked with creating a simple calculator capable of performing fundamental arithmetic operations: addition, subtraction, multiplication, and division. This practical exercise enabled me to solidify my understanding of HTML forms for user input, JavaScript functions for computation, and CSS for styling the calculator's interface. This project also honed my problem-solving skills, as I had to troubleshoot and debug the code to ensure its smooth functionality.

Personal Portfolio Website:

The culmination of the web development module was the creation of a personal portfolio website. Building on the knowledge gained during the week, I undertook the exciting task of crafting a digital showcase of my skills and accomplishments. This endeavor not only reinforced my understanding of HTML and CSS but also prompted me to explore the principles of responsive web design, ensuring that the website would seamlessly adapt to various screen sizes and devices.
